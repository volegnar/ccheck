# ccheck - (version 1.0)
Very basic command-line crypto price checker in EUR and USD

Available currencies:
- BTC
- ETH
- SOL
- LTC
- XMR
- DOGE
- ADA

Convertible to:
- EUR
- USD

*prices will appear in random colors*
